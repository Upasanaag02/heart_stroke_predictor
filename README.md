# 🫀 Heart Stroke Predictor

A machine learning web app that predicts the risk of heart 
disease based on patient health data.

## 🌐 Live App
👉 https://heart-strokepredictor.streamlit.app

## 📌 Overview
- Predicts heart disease risk using a KNN (K-Nearest Neighbors) model
- Takes patient health inputs and returns risk prediction instantly
- Deployed as an interactive web app using Streamlit

## 🛠️ Tech Stack
- Python
- Scikit-learn (KNN Model)
- Pandas, NumPy
- Streamlit (Web App)
- Matplotlib (Visualizations)

## 📊 Model Performance
- Algorithm: K-Nearest Neighbors (KNN)
- Accuracy: **88%**
- Preprocessing: Standard Scaling
- Dataset: Heart Disease Dataset

## 🚀 Run Locally
git clone https://github.com/Upasanaag02/heart_stroke_predictor
cd heart_stroke_predictor
pip install -r requirements.txt
